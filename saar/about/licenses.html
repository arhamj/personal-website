Bhagavad Gita — Sanskrit text & transliteration

Public domain text · Source dataset on GitHub

The Devanagari Sanskrit and IAST transliteration shipped with Saar were sourced from the vedicscriptures/bhagavad‑gita dataset compiled by Pt. Prashant Tripathi.

The Sanskrit text of the Bhagavad Gita is an ancient work and is in the public domain worldwide. Mechanical IAST transliteration of public‑domain Devanagari is likewise treated as public domain. English translations, commentary, chapter summaries, and all other text in Saar are original work, not derived from the upstream dataset.

If we have missed an attribution, please leave a note in the App Store review for Saar and we will correct it.
